# 🚀 ROJGAAR 2.0

ROJGAAR 2.0 is a web platform that connects workers and employers. It allows users to post jobs, search for work, and manage opportunities easily .

 Features
* Worker & Employer Login
* Job Posting
* Job Search
* Dashboard for Users
* Basic Map Integration

 Tech Stack
* **Frontend:** HTML, Tailwind CSS, JavaScript
* **Backend:** Python (Flask)
* **Database:** SQLite (SQLAlchemy)

 Setup

```bash
git clone https://github.com/your-username/rojgaar-2.0.git
cd rojgaar-2.0
pip install -r requirements.txt
python app.py
```
